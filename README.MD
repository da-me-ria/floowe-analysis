# Floowe Product Analysis – Technical PM/BA Task

This repository contains the analysis of the Floowe application as part of the recruitment task for AI-Native Technical Project Manager / Business Analyst.

## Repository Structure

* `/analysis` – product analysis and UX review
* `/flows` – key user flows and process descriptions
* `/event-storming` – AS-IS and TO-BE domain events
* `/features` – proposed product features
* `/architecture` – system architecture overview

## AI Support

AI-tools were used to assist with:

* structuring product analysis
* identifying domain events
* accelerate user flow modeling
* generating architecture diagrams
* drafting documentation

All outputs were reviewed and refined manually.
